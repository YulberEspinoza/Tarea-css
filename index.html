  • ¿Cuál es el propósito de CSS?

    • Css nos ayuda a que nuestra pagina adquiera un aspecto mas agradable en función de los que querramos compartir. Desde darle color, formato, tamaño y demas cualidades.

  • ¿Cuáles son las tres formas de insertar CSS en tu proyecto?

      • Hay tres formas en las que podemos insertar nuestro proyecto en Css

    • Css interno: Desde el propio código de html podemos poner una etiqueta de Style entre entre signos de mayor y menor como cierre..
    • Css Inline: Se incerta una etique en la línea de código del tipo Style:
    • Css externo: Cuando se crea un archivo externo y se víncula al html con la etiqueta link.
  • Escribe un ejemplo de una regla CSS que daría texto rojo a todos los elementos "P"

    • Para poder tener el resultado que nos pide la pregunta necesitamos poner una etiqueta de color y con el respectivo formato

    color: red
